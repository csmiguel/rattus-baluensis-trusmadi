# Population genetics of *Rattus baluensis*
Study about the genetic diversity and population structure in the mountain rat *Rattus baluensis*, endemic to northern Borneo, using intron markers.

We genotyped the sequencing data with [tidyGenR](https://github.com/csmiguel/tidyGenR), an R package developed for genotyping multilocus amplicons.
